### Langkah-Langkah Instalasi

Ikuti langkah-langkah di bawah ini di Terminal Anda:

1.  **Kloning Repositori:**
    ```bash
    git clone [https://github.com/muhammadrohendi112-prog/Garaca.git](https://github.com/muhammadrohendi112-prog/Garaca.git)
    cd Garaca
    ```

2.  **Instal Dependensi:**
    ```bash
    # Jika menggunakan Node.js/NPM
    npm install
    
    # Jika menggunakan Python/PIP
    pip install -r requirements.txt
    
    # Jika menggunakan Composer (PHP)
    composer install
    ```

3.  **Konfigurasi Lingkungan:**
    * Buat salinan dari file konfigurasi (`.env`):
        ```bash
        cp .env.example .env
        ```
    * Edit file `.env` dan masukkan kredensial database dan kunci API yang diperlukan.

4.  **Migrasi Database (Opsional):**
    ```bash
    # Contoh untuk Laravel/Django
    php artisan migrate 
    ```
